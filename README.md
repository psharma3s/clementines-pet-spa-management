# clementines-pet-spa-management
A full-stack pet spa management system build with Java, Spring Boot, PostgreSQL, and React.


Clementine's Pet Spa Management System is a full-stack learning project designed to explore professional software engineering practices through the development of a modern web application.

The project serves as a hands-on environment for learning Java, Spring Boot, REST APIs, PostgreSQL, React, Git, and modern development workflows while progressively transforming a simple business website into a complete full-stack application.

The focus of this repository is understanding software architecture and development practices rather than rapidly completing features. As new concepts are learned, the application will evolve alongside that knowledge.
